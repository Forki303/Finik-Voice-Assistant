# Finik-Voice-Assistant
Головой ассистент написанный на Python

---

# Доступные команды
1) Финик, дальше. Финик следующая -- Перелистование музыки вперёд
2) Финик, назад. Финик предыдущая -- Перелистование музыки назад
3) Финик, стоп -- Остановить музыку
4) Финик, продолжи. Финик плей -- Продолжить воспроизведение музыки
5) Финик, выключи компьютер. Финик, выключи ПК -- Выключение ПК
